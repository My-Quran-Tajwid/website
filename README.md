# My Quran Tajwid Website

The My Quran Tajwid Website is a modern, responsive website built with [Astro](https://astro.build) and styled using [Tailwind CSS](https://tailwindcss.com).

## Getting Started

To get started with the project, follow these steps:

```bash
npm install
npm run dev
```
